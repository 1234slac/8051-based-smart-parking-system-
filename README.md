# 8051-based-smart-parking-system-
8051-based smart parking system with automated gate control using Proteus simulation and motor driver interfacing
##  Overview

This project presents a microcontroller-based smart parking system using the AT89C51 (8051) that automates vehicle entry and exit through controlled gate operation. The system uses input signals representing entry and exit sensors to detect vehicle movement and processes these signals using embedded logic.

Based on sensor inputs, the microcontroller controls a motor driver to operate a gate mechanism, allowing vehicles to enter or exit the parking area. The system also maintains a basic parking slot count and provides visual indication of system activity.

The design is implemented and tested using Proteus simulation, with the control logic developed in Embedded C using Keil µVision. This project demonstrates the integration of sensing, processing, and actuation in an embedded system for real-world automation applications.
